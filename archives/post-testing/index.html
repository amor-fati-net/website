久々のwordpressです． 投稿テストしてみます．

改行はどう処理されるのか． テストしてみた．結果は…

  • Enterだけだと別のpになる．
  • Shift+Enterならbrするのか！なるほど！これは素晴らしい．

数式がきれいに表示されるプラグインあるかな． あってほしいな．

探したら幾つか見つかった． とりあえず一番最近更新された LaTeX for Wordpress を入れてみた．

とりあえずインラインで二次関数 ( f(x) = a x^2 + b x + c ) を書いてみる．

で，ちょっと込み入った式として，ガンマ関数はこうなるらしい． 式を１行分で表示させる．

[ \Gamma(z)=\int^{\infty}_{0}t^{z-1}e^{-t},dt\qquad(\mathcal{R} {z}>0) ]

素晴らしい！けど，数式が文字の大きさ的に小さい． これはCSSで何とかなるハズなので，問題ないだろうと思う．

見た目について色々と対処を試みた結果… まずブラウザによって表示される大きさが異なることが判明． Macの Chrome, Safari では小さく見えて，Android のデフォルトブラウザ，LinuxのChrome, Firefox 7.0 では丁度よい大きさ． Winの IE8 ではちょうど良く表示されるが，単純にフォントがダサい．

まぁ大した問題ではないと，あまり気にしないことにしよう．
